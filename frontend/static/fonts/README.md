# Fuentes y modo local

Esta release pública de PlumA no descarga fuentes ni llama a Google Fonts, CDN u otros servicios externos durante la carga de la interfaz.

La hoja de estilos usa pilas tipográficas del sistema para mantener la instalación local y evitar dependencias online. Si en el futuro se incorporan fuentes autohospedadas, deberán añadirse al repositorio con su licencia de redistribución correspondiente y sin scripts de descarga en la release de usuario final.
